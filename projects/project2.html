Project2

Overview

RubyおよびDXRubyライブラリを用いてチームで課題解決に取り組むRuby合宿2024春に参加し、2Dシューティングゲームの作成を行いました。 作成したゲーム「リサイクルオデッセイ」は、未就学児や海外からの留学生にとって難しいとされるゴミの分別を楽しく学べることを目的としています。 チームには中国からの留学生が在籍しており、開発やディスカッションは英語で行いました。

Tools / Environment

開発・環境および利用したツールについて

  • OS:Windows 11 Home 64bit
  • Ruby:3.0.3 (via RubyInstaller)
  • DXRuby:1.4.7
  • エディタ:Visual Studio Code
  • GitHub:ソースコードの共有およびバージョン管理で使用

Demonstration

実際のプレイ動画です。

操作方法

Z: 弾1（燃えるゴミ袋）を発射
X: 弾2（燃えないゴミ袋）を発射
降ってくるゴミを正確に撃ち落とそう！
スコアによってエンディングが変化します
動画は最高スコアのエンディングです

Code Structure

ゲームのソースコード構成について簡単に説明します。

project-root/
├── images/               # ゲーム内で使用する画像リソースを格納
├── sounds/               # ゲーム内で使用するサウンド（効果音やBGM）を格納
├── scenes/
│   ├── base.rb           # シーン共通の基本クラス（継承元）を定義
│   ├── ending.rb         # ゲーム終了時の演出や処理を定義
│   └── gameover.rb       # ゲームオーバー時の表示や処理を定義
│           ︙
├── bullet.rb             # 弾の挙動や当たり判定を管理するクラス
├── enemy.rb              # 敵キャラクターの挙動や状態を管理するクラス
├── ememyspawner.rb       # 敵の出現管理処理を行うクラス
├── levelmanager.rb       # ステージや難易度の進行管理を行うクラス
├── player.rb             # プレイヤーキャラクターの動作や状態を管理するクラス
└── main.rb               # アプリケーションのエントリーポイント（初期化・起動処理）

Conclusion

本合宿でのチーム開発にはチームリーダーとして参加した。参加の目的であったRubyおよびDXRubyライブラリを用いたゲームを作成することが達成できた。またプロジェクトを進めるにあたりGithubを用いたバージョン管理についても学習することができた。英語でのディスカッション、開発の経験も積むことができたので非常に実りのある合宿となった。
一方作品に目を向けると現段階では様々な改良の余地がある。例えばゴミの種類を増やしたり地区町村によって異なるゴミの分別やゴミ袋の色を学べるような仕組みを取り入れる事が挙げられる。 また留学生向けのUIとして多言語向けに改良するなども課題として挙げられる。今後UI、レベルデザインなどの改良に取り組みたい。
